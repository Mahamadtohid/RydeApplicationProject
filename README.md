# RydeApplicationProject
Backend for Ryde — a microservices-based ride management system built using Spring Boot. This repository contains all backend services including booking, user, location, and more.

This repository contains the backend architecture of the Ryde Application, a microservices-based ride management platform developed using Spring Boot and related technologies.
It serves as the core backend system that powers the Ryde ecosystem — handling users, bookings, payments, locations, and inter-service communication.
